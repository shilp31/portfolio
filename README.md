# Portfolio Website

A simple, modern, and responsive portfolio website to showcase your work, skills, and projects.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Easy to Customize**: Simple HTML, CSS, and JavaScript structure
- **Smooth Scrolling**: Elegant navigation between sections
- **Mobile Menu**: Hamburger menu for mobile devices

## Sections

1. **Home**: Hero section with introduction
2. **About**: Brief introduction about yourself
3. **Skills**: Display your technical skills
4. **Projects**: Showcase your work and projects
5. **Contact**: Links to get in touch

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content:
   - Edit `index.html` to update text, links, and content
   - Modify `styles.css` to change colors and styling
   - Update `script.js` if you want to add more interactivity

## Customization

### Personal Information
Edit the following in `index.html`:
- Your name in the hero section
- About me description
- Skills and technologies
- Project details
- Contact information (email, GitHub, LinkedIn)

### Colors
Update the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    /* Add more color variables */
}
```
